<p>
<b>ANALYZE</b> 
<ol type="1">
  <li>Research and analyze the target websites to understand their HTML structure, image tags, and any relevant attributes. </li>

  <li>Handle any error handling, pagination, or authentication requirements.</li>
</ol>  
</p>

<p>
<b>EXECUTE</b> 
<ol type="1">
